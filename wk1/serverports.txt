What is the definition of server port?

A port is a logical construct that identifies a specific process or a type of network service. 
A port is identified for each transport protocol and address combination by a 16-bit unsigned number, 
known as the port number.

What is server port range?

0 to 65535

How many bits does number above represent?

16 bits
